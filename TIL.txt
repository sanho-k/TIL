2022.01.16(일)
issue와 brance, merge!
Issue : 내가 작업할 것, 오류, 기능 추가, 버그, 개선 제안, 작업 단위 등등
Brance : 작업 공간 분리하는 것 각자 공간에서 작업 가능
Merge : 각자 작업한 내용(brance)들을 합치는 것  
Tracking = 원격repo와 로컬repo의 특정 브랜치를 연결(트랙킹)해주는 것
Merge conflict : 다른 브랜치에서 같은 파일의 동일한 부분을 수정할 시 생기는 오류!
두 브랜치를 머지해주고 겹치는 부분을 수정해준후 커밋하면 오류 해결
브랜치 작업을 완료 후 병함(머지)을 완료했을 시 완료한 브랜치들은 삭제 = 관리 용이
Check-out : 작업할 브랜치로 이동(선택)하는 것
